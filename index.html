Installation Troubleshooting

If your installation failed, find the return/exit code below to resolve the issue.

Return Code Meaning / Error How to Fix
0 Success The application installed successfully. No action required.
1603 Fatal error during installation Common Windows Installer error. Run the installer as an Administrator or check if a previous version is pending a reboot.
1618 Another installation in progress Another program is currently installing. Wait for it to finish, restart your PC, and try again.
1001 Insufficient Disk Space Free up at least 500MB of space on your main drive and try again.
